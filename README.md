# Jukhunant — Alsace 2026 🍷

Vacation planning site for the **Jukhunant** crew (Juup, Anant, Sukhuma) for our trip to Alsace, France, **17 – 21 July 2026**.

## 🌐 Live site

Once GitHub Pages is enabled, the site will be available at:

> https://juuphietbrink.github.io/jukhunant/

### Enabling GitHub Pages

1. Go to **Settings → Pages** in this repository.
2. Under *Build and deployment*, set **Source** to `Deploy from a branch`.
3. Select branch **`main`** and folder **`/ (root)`**, then **Save**.
4. Wait ~1 minute and refresh — the URL above should be live.

## 📁 Structure

```
.
├── index.html   # The single-page itinerary
├── styles.css   # Alsace-themed styling
├── script.js    # Packing checklist persistence + countdown
└── README.md
```

Pure static site — no build step required.

## ✏️ Editing the plan

Just edit `index.html` and push to `main`. GitHub Pages will redeploy automatically.
